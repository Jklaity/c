# Mentality of designing
设计思路：
由题可知，我们需要找到不同的型号以及该型号下所对应的剂量和次数，在利用编程语言时，可采用枚举的方法将其一一例举出来，再让计算机统计一共的结果即可，在这里有很多种方法实现，为了方便观察，我本次实验按照10-180的注射次数排序，一一例举出所有可能的结果，利用for循环和条件判断语句if即可简单完成。
